# Hourly Energy Consumption Forecast

## 📌 Objective
This project focuses on forecasting hourly energy consumption using Machine Learning techniques. The goal is to analyze historical power consumption patterns, build predictive models, and forecast future energy demand to support efficient energy management and planning.

---

## 📚 Topics Covered
- Exploratory Data Analysis (EDA)
- Data Cleaning and Preprocessing
- Feature Engineering
- Time Series Analysis
- Machine Learning for Forecasting
- XGBoost Regression
- Model Evaluation
- Future Energy Demand Prediction
- Streamlit Deployment

## ▶️ How to Run

### Run the Jupyter Notebook
1. Download or clone the repository.
2. Open Anaconda Prompt.
3. Navigate to the project folder:

```bash
cd path_to_project_folder
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open `Hourly_Energy_Consumption_Forecast.ipynb`.
6. Run all cells sequentially.

### Run the Streamlit Application

1. Open Anaconda Prompt.
2. Navigate to the project folder:

```bash
cd path_to_project_folder
```

3. Run the Streamlit app:

```bash
streamlit run app.py
```

4. The application will open automatically in your browser.

## 📊 Dataset Information

The dataset contains historical hourly energy consumption records from the PJM Interconnection power grid.

### Features

| Column Name | Description |
|------------|-------------|
| Datetime | Date and time of energy consumption |
| PJMW_MW | Hourly energy consumption in Megawatts (MW) |

### Dataset Usage
- Energy Demand Analysis
- Time Series Forecasting
- Trend and Seasonality Detection
- Future Consumption Prediction
